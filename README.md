# 📰 Fake News Detection System (TruthLens AI)

## 📌 Problem Statement
In today’s digital world, misinformation spreads rapidly through social media and online platforms. It becomes difficult for users to identify whether a news article is real or fake.  

This project aims to build an **AI-based Fake News Detection System** that analyzes news content and provides insights such as authenticity, confidence score, source credibility, and live verification.

---

## 🚀 Project Overview
TruthLens AI is a web-based application that allows users to:

- Paste news text or URL
- Automatically extract article content
- Detect whether the news is Fake or Real
- View confidence score
- Check source credibility
- Perform live verification using news APIs
- Visualize risk using a circular risk meter

---

## 🧠 Features

✔ Fake / Real News Prediction  
✔ Confidence Score

✔ Source Credibility Check  
✔ Live News Verification (NewsAPI)
